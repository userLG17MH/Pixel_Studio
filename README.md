# Pixel_Studio
1. Asset & Document Management: The interface includes tabs for managing Assets (images), Documents (PDFs), and Presentations.

2. Image Compression Engine: A functional tool to reduce image file sizes with live previews, custom target sizes (in KB), and quality sliders.

3. Pixel Art Generator: A creative module that converts uploaded images into pixel art. It allows you to adjust block sizes, apply color palettes (including a custom "v5 Acid" palette), and toggle effects like grid lines and dithering.

4. Active Input Stream: A drag-and-drop "Input Stream" zone that handles multiple file uploads and maintains an active processing queue.

5. Bidirectional conversion (Image ↔ Excel) Pixel-level processing Performance handling (chunking + progress) UX (drag-drop, logs, stats, preview)
